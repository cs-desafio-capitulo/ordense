# Ordense

É o repositório que trata do serviço responsável por lidar com os pedidos (que vão possuir produtos), realizar suas validações e seguir para o serviço de pagamento.

### Stack utilizado
#### Serviços
* Heroku (versão free)
* Mlab (versão free)
* Github

#### Definições técnicas
* Node (versão 8.9.3 LTS)
* Git
* Babel
* Mongoose
* Jetkins
* Express
